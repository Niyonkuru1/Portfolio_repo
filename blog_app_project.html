The Blog Website

To publish and manage blogs

I developed frontend of this website using vanilla javascript, html and css.

I then develope its API using Node.js, Express, Mongo, Jwt in the backend.

The admin can login with correct credential and be redirected to the his / her dashboard. In there he/she can create the blog, edit the blog, read all blogs and delete a certain blog.

The API are integrated into the front-end using AJAX calls.
